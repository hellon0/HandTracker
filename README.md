# HandTracker
Hand tracker made in python using pytorch and opencv

Currently a work in progress. The MediapipeTracker.py program uses a pre-existing hand tracking algorithm, serving as a goal for my project.

MyHandTracker.ipynb is the current code for the hand tracking model, both initialization and training. The training loop is complete, but the architecture of the Network is chosen based upon rough guesswork, and will likely need to be tweaked. As of right now, my goal is to test the Network and see how accurate it will get without making adjustments to the architecture.
